؟وش هالبرنامج؟What is this?

Mac Monitor أداة سطر أوامر (CLI) بسيطة تسوي لقطة سريعة لكل العمليات اللي تشتغل بجهاز macOS، تصنفها وتشرحها لك بكلمات واضحة، وتعطيك خيار توقف أي عملية تقيلة بنقرة.

تستخدم ذكاء اصطناعي (AI) مجاني تماماً من Groq — يمديك تسأله 14,400 سؤال عن العمليات كل يوم بدون ما تدفع ريال.

ولأننا نعتمد على "سلسلة احتياط" ذكية (Cache → Whatis → AI)، النتائج تجيك فوراً في الزيارة الثانية.

Mac Monitor is a terminal tool that takes a snapshot of every running process on your Mac, categorizes them, explains what each one does in plain English, and lets you kill any heavy process instantly.

It uses a free AI (Groq) — you get 14,400 free descriptions per day, no credit card required.

With its smart fallback chain (Cache → Whatis → AI), descriptions appear instantly on the second run.

نصب Python والمكتبات، افتح حساب في Groq، وخذ مفتاح API مجاني بدون بطاقة ائتمان.

Install Python dependencies, create a free Groq account, and get your API key — no credit card needed.

2

شغل الأمر

Run the command

شغل python mac_monitor_groq.py، بتظهر لك لقطة فورية لكل العمليات.

Run python mac_monitor_groq.py and see an instant snapshot of every process.

3

فهم وشوف اللي يستهلك

See what's eating your CPU

الجدول مقسم ٣ أقسام: البرامج الآمنة (🟢)، خدمات الخلفية (🟡)، والنواة (🔴). الأوصاف تتعبأ في الخلفية.

Processes are sorted into 3 sections: safe apps (🟢), background services (🟡), system core (🔴). Descriptions fill in live.

4

أوقف اللي تبغى

Kill what you want

اكتب رقم العملية وضغط Enter. التطبيقات فقط القابلة للإيقاف. النواة محمية.

Type a process number and hit Enter. Only apps and services — system core processes are protected.

Smart protection

يقسم العمليات لـ 3 فئات. فقط الخضراء والصفراء تقدر توقفها. الحمراء محمية ولا يمكن قتلها.

3 categories. Only green and yellow can be killed. Red (system core) is protected.

🛡️سلسلة الاحتياط (فشل ← جرب اللي بعده)Fallback chain (fail → try next)

ما يعتمد على مصدر واحد. لو انقطع الإنترنت أو تعطلت API، يشتغل بدون نت:
Never depends on a single source. If the internet drops or the API fails, it still works offline:
1 SQLite Cache — أوصاف محفوظة من الجلسات السابقة SQLite Cache — saved from previous sessions فوري instant
2 Hardcoded Overrides — أسماء معروفة مسبقاً (VS Code, opencode...) Hardcoded Overrides — known names (VS Code, opencode...) فوري instant
3 Path Extraction — اسم التطبيق من مسار التثبيت Path Extraction — app name from install path فوري instant
4 Whatis — دليل macOS المدمج (لخدمات النظام) Whatis — macOS built-in manual (for system daemons) سريع fast
5 Groq AI — ذكاء اصطناعي سحابي مجاني (14,400 وصف/يوم) Groq AI — free cloud AI (14,400 descriptions/day) إنترنت online
6 Generic Fallback — وصف عام: "macOS process: {name}" Generic Fallback — "macOS process: {name}" آخر ملاذ last resort
